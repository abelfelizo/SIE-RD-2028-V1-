# SIE 2028 — Sistema Inteligencia Electoral v9.4

## CÓMO USAR (2 opciones)

### OPCIÓN 1 — SIN SERVIDOR (más fácil) ✅
Abre directamente:
  sie2028_standalone.html

Doble clic → abre en el navegador → funciona inmediatamente.

### OPCIÓN 2 — CON SERVIDOR
En la carpeta del proyecto:
  python3 -m http.server 8080

Luego abre: http://localhost:8080
